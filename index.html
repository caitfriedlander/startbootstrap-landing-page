Cait Friedlander

Full-Stack Web Developer & Designer

I am a web developer and lifelong geek who recently had the chance to embrace my love for technology by attending a Web Development Immersive program to learn new technologies and refine my skills, after several years in the visual arts- illustration, graphic design, and costuming for film, TV, and private clients. I look forwards to using what I learned in the arts - creativity, flexibility, and client driven workflow - to solve problems and write great code for you.

Technical skills include: HTML5, CSS3, JavaScript, Ruby, Python, Node.Js + Express, Rails, Flask, Angular, jQuery, Bootstrap, Materialize, PostgreSQL, MongoDB, Mongoose, SaSS, AWS, Git/GitHub, and Adobe Creative Suite.

Snake!

A new twist on the classic game! Using a combination of Javascript, math, and childhood nostalgia I recreated the classic game where you steer your snake around the screen, using the arrow keys, collect apples to earn points, and try to avoid colliding with yourself or the walls.

Technologies Used: HTML, CSS, JavaScript and jQuery.
Deployed on Github Pages.

Moodboarder

A Full-Stack CRUD app for making mood boards- a helpful tool for design work and brainstorming.

Technologies Used: Ruby on Rails, HTML, CSS, SaSS, Bootstrap, JavaScript, jQuery, Paperclip, and AWS.
Deployed on Heroku.

The Gif That Keeps On Gif-ing

A fun app that asks users to answer questions about their lives and personalities and then translates their answers into a sharable, gif-filled timeline using the Giphy API. Technologies Used: MongoDB, Node.js + Express, JavaScript, jQuery / AJAX, Google OAuth 2.0, Materialize, HTML, CSS and the Giphy API.
Deployed on Heroku.
